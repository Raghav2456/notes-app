# React Notes App

A simple Notes Management Application built using React.js and Vite. This project allows users to create and manage notes through a clean and interactive interface. It demonstrates the fundamentals of React, including functional components, state management, form handling, and dynamic rendering.

## Features

- Add new notes
- Delete existing notes
- Real-time UI updates using React State
- Simple and responsive user interface
- Fast development setup with Vite

## Tech Stack

- React.js
- JavaScript (ES6+)
- Vite
- CSS

## Installation

1. Clone the repository

git clone <repository-url>

2. Navigate to the project directory

cd 13-notes-app

3. Install dependencies

npm install

4. Start the development server

npm run dev

## Learning Objectives

This project demonstrates:

- React Functional Components
- useState Hook
- Form Handling
- State Management
- Dynamic Rendering
- Event Handling in React
